# Initial 
 git clone <repository> 

wenn das meine Repo ist 
 git clone <ssh repo> -> git@github.com:dev-foo-bar/examples.git
 git clone  git@github.com:dev-foo-bar/examples.git

wenn ich anonym Clone 
 git clone <https> -> https://github.com/dev-foo-bar/examples.git
 git glone https://github.com/dev-foo-bar/examples.git


# File / Änderung im File hinzufügen 

 Änderungen sind: neue Datei, Datei verändert, Datei löschen 

 git add <Name des Files> 
 git add Testfile 

# Mehrere Änderungen hinzufügen 
 git add .

# Änderungen finalisieren 

 git commit

 oder 

 git commit -m "nachricht" 


# Änderungen auf die Remote-Repository "schieben"

 git push 

# Änderungen von der Remote-Repository "holen"

 git pull
